## Setup

### Web3 (backend)

- Clone the project and change directory to web3

    ```shell
    git clone https://github.com/oggy107/AuthentiScan.git
    cd AuthentiScan/web3
    ```

- Install dependencies

  ```shell
  npm install
  ```
